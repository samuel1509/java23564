Inicializar un dir git y subirlo a github:
=========================================
- instalar git https://git-scm.com/downloads
- crear cuenta en github github.com/
- crear carpeta donde van a programar el integrador, abrimos vsc en la carpeta, pego mi proyecto y crear archivo index.html 
- en vsc terminal -> new terminal 
- verificar version git -v
- git init  // inicializa el proyecto en git 
- git add . //agrega todos los archivos nuevos al área de preparación
- git commit -m 'comentario' //sube los archivos del area preparación a repo local
- github crear repositorio 
- git remote add origin https://github.com/sbsanchez21/intFront-2023-23564.git
- git push -u origin master

Subir archivos modificados y nuevos al repo remoto:
===================================================
- git add .
- git commit -m 'link a oradores'
- git push -u origin master


Publicar proyecto online en github pages:
=========================================
- settings -> pages
- branch -> master save
